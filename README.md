# Wshop

Wshop is a building shop app. The app has 2 separate modules (admin panel and client shop).

### Technologies:

 Spring boot
 
 Spring webmvc
 
 Spring security
 
 Spring data

 Spring mail
 
 Thymeleaf
 
 Ckfinder and CkEditor
